# Currencies

Currencies are the primary reward system in Alchemize.

As you build projects and track your work, you'll earn currencies based on the theme your project belongs to. These currencies can be spent in the shop on theme-exclusive rewards, or converted into Potion Mix through the Trade Portal.

---

## Earning Currency

For every hour tracked on an approved project, you earn **1 unit of the currency associated with that project's theme.**

Season 1 currencies:

| Theme         | Currency   |
| ------------- | ---------- |
| Indie Gamedev | Redstone   |
| No Internet   | Glowstone  |
| Endless       | Aqua Regia |

Examples:

* 15 hours on an Indie Gamedev project → 15 Aqua Regia
* 20 hours on a No Internet project → 20 Glowstone
* 8 hours on an Endless project → 8 Redstone

The more you build, the more currency you earn.

---

## Theme Exclusive Rewards

Each theme has rewards that can only be purchased using that theme's currency.

### Aqua Regia

Earned from the **Indie Gamedev** theme.

Can be used to purchase Indie Gamedev-exclusive rewards.

### Glowstone

Earned from the **No Internet** theme.

Can be used to purchase No Internet-exclusive rewards.

### Redstone

Earned from the **Endless** theme.

Can be used to purchase Endless-exclusive rewards.

---

## Currency Values

Currencies also have a value when converted into Potion Mix.

| Currency   | Potion Mix Value |
| ---------- | ---------------- |
| Redstone   | 4                |
| Glowstone  | 4.5              |
| Aqua Regia | 5                |

These values are used by the Trade Portal when converting currencies into Potion Mix.

---

## Potion Mix

Potion Mix is Alchemize's universal currency.

Unlike theme currencies, Potion Mix can be used to purchase general rewards that are not tied to a specific theme.

Potion Mix is created by combining theme currencies through the Trade Portal.

**Potion Mix cannot be used to purchase theme-exclusive rewards.**

If you want a theme-exclusive item, you'll need to earn the matching currency.

---

## Why Multiple Themes Matter

You can participate in only one theme if you'd like.

However, participants who build projects across multiple themes unlock more opportunities to convert currencies into Potion Mix and take advantage of the Mixing System.

More ingredients often lead to better potions.
